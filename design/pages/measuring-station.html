Measuring Station

Usage

This station can be used to do the measurements with electronic instruments such as Oscilloscopes, Signal Generators, etc... They also can do some prototyping works with breadboards in here

Capacity: 1-3 students per table
Number of Stations: 2

Tools and Accessories

Machines / Tools per Station
  • Computer + Monitor
  • Digital Multimeter
  • Power Supply
  • Digital/Analog Oscilloscope
  • Signal Generator
  • Digital Logic Analyzer
Components
  • Jumper Cables (MM, MF, FF)
  • Single Core Wire sets
  • 2 x Breadboards
  • 6 x Alligator clip cables
  • 6 x Banana plug cables
  • 3 x DC Barel Jack cable
  • 1 x USB cables (type D, micro, mini)
  • 1 x Serial port cable
  • 1 x Parallel port cable
  • 1 x Ethernet cable
  • 2 x Logic level converter
  • 1 x RS232 to TTL converter
  • 1 x USB to TTL converter
  • 1 x HDMI extension cable
Other Equipment (Optional)
  • USB Extension Hub
  • Magnifier Glass

Special Notes

  • Some number of resistors, capacitors, diodes should be provided in the table.
  • An instruction manual should be provided about the instrument (via online)
