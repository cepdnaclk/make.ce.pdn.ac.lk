MakerBot Replicator User Guide

If you are using MakerBot Replicator 3D printer in the department, please read this documentation first.

Things you need to consider before start a 3D Print:

  • Check the time estimation of the design at first. Add 25% overhead + 15 mins to the estimated time and see wether you can complete the 3D print before 5 PM (or whatever time that MakerSpace will be closed)
    • Estimated time : 2 hours
    • Total time after overhead: 2 hours and 25 mins
    • If the current time is on or before 2:15 PM, you can start 3d printing. Otherwise, you need to start it in the next day morning.
  • Since the materials are limited, please try your best to reduce the volume of your design.
  • Make sure to enable Print support structure option in the Creality Slice software, if your design has any overhangs / bridges.
